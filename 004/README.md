# Algoritmo de Cadastro e Pesquisa de Hóspedes

Monte um algoritmo em que o usuário poderá cadastrar e pesquisar hóspedes.

## Menu de Opções

O algoritmo oferece um menu com três opções:

1. **Cadastrar**: Permite informar um nome de hóspede e gravá-lo em memória (máximo de 15 cadastros). Se o limite for atingido, exibe “Máximo de cadastros atingido”.
2. **Pesquisar**: Permite informar um nome. Se encontrado exatamente, exibe “Hóspede (nome) foi encontrado no índice (índice)”. Caso contrário, “Hóspede não encontrado”.
3. **Sair**: Encerra o algoritmo.

O algoritmo permite operações repetidas até a escolha da opção 3.
