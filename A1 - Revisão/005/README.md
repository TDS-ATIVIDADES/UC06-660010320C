# Programa Java - Controle de Quartos de Hotel

Monte um programa Java que mostre na tela em formato de tabela os quartos ocupados e os desocupados.

## Especificações

- O hotel possui 4 andares, cada um com 3 quartos.
- O usuário registra os quartos ocupados, informando o andar (1 a 4) e o número do quarto (1 a 3) para cada ocupação.
- Após cada registro, o programa pergunta: “Deseja informar outra ocupação? (S/N)”.
- O usuário pode registrar quantos quartos quiser até responder "N".
- Em seguida, o programa exibe uma tabela onde:
  - O primeiro andar é o inferior e o último é o superior.
  - Quartos ocupados são marcados com "X".
